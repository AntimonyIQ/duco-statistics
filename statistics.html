About Duino-Coin

Duino-Coin is an open-source cryptocurrency that can be mined with low-end computers, smartphones, Arduino boards and many other platforms. It uses SHA algorithm for blocks, similar to the one used by Bitcoin and can be mined with many platforms. If you're interested read official Getting Started guide.
