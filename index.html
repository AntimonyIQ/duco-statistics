 About Duino-Coin

Duino-Coin is an innovative cryptocurrency project. It uses SHA algorithm for blocks, similar to the one used by Bitcoin and can be mined with many platforms, from regular computers to Arduino boards. If you're interested in mining, you should check out official Getting Started guide.
